# socketio-server

## Introduction
Socket.io real-time server for <a href="https://github.com/SimoneC03/socketio-chat-inertia" target="_blank">socketio-chat-inertia</a>

## Installation

```bash
git clone https://github.com/SimoneC03/socketio-server
cd socketio-server
npm install
```
## Starting server
```bash
npm run start
```
